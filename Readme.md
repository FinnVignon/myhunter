<h1 align="left">My Hunter :  Mario Hunter</h1>

###

<p align="left">Mario Hunter is a Duck Hunt inspired shooter where you try to stop runners from escaping their pipes.</p>

###

<h2 align="left">Usage :</h2>

###

```bash
# compiles ./my_hunter from the .c files
make
#launch program
./my_hunter
# or if you need help
./my_hunter -h
```

###

<h2 align="left">Controles</h2>

###

<p align="left">- Left Click: shoot runners on screen.<br>- Space: continue past the tutorial screen.<br>- Escape: return to the main menu while playing.<br>- Menu Buttons: click Start to play, Exit to quit the game.</p>
